# Changelog

This repository contains multiple crates. Each crate has its own changelog:

- [notify](notify/CHANGELOG.md) - Cross-platform filesystem notification library
- [notify-debouncer-full](notify-debouncer-full/CHANGELOG.md) - Debouncer for notify with full event handling
- [notify-debouncer-mini](notify-debouncer-mini/CHANGELOG.md) - Minimal debouncer for notify
- [notify-types](notify-types/CHANGELOG.md) - Types used by the notify crate
- [file-id](file-id/CHANGELOG.md) - Utility for reading inode numbers (Linux, MacOS) and file IDs (Windows)
